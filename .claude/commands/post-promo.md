# /post promo — Promo kolekce s mint linkem

Týdenní promo post. Vždy obsahuje mint link, rarity highlight nebo rewards info.

## Usage
```
/post promo --collection <name>
/post promo --collection <name> --profile <handle>
```

`--collection` (required): `pixelonkas`, `sykora`, `pixel`, `synthi`
`--profile` (optional): `pixelonkas`, `marekozor` — pokud vynecháno, zobraz oba

## Promo rotace
Dodržuj pořadí: `PIXELONKAS → SYKORA → $PIXEL → SYNTHI → (opakovat)`

## Step 1 — Vygeneruj post

### @PixelonKas — X
- Short tweet, max 220 znaků + mint link + hashtags
- Alternativně medium (220–280 znaků) — variovat
- Vždy obsahuje: mint link nebo trade link pro danou kolekci
- Hashtags: `#Kaspa #NFT` + kolekce-specifické

### @PixelonKas — Instagram
- Krátký nebo střední caption
- Popis co kolekce je, proč mint, co získáš
- Prázdný řádek před hashtag blokem
- Hashtags (povinné): `#Kaspa #KRC721 #NFT #GenerativeArt #PixelArt #CryptoArt #BlockchainArt`

### @marekozor — X
- Osobní doporučení — proč on věří v tuto kolekci
- Krátké, max 200 znaků
- Vždy přidat: `opensea.io/marekozor` + mint link kolekce
- Hashtags: `#GenerativeArt #Kaspa`

## Step 2 — Doporučený čas
- @PixelonKas: `14:00–16:00 UTC`
- @marekozor: `09:00–11:00 UTC`

## Step 3–5 — Schválení a posting
Stejný flow jako `/post` Step 4–6.
