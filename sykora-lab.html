★ Sýkora = 14 abstract black-fill elements × 4 rotations, fully straight-edge. Each element is built from a base shape plus characteristic +/− triangles (chamfers, accents, V-cuts, bridges) — Triangle sliders modulate them.
Triangle size
—
Size of the +/− triangles built into every element. Above ~1.2 they exceed the cell and overlap neighbours.
Triangle balance
—
Low = only negatives (white carves). High = only positives (black accents). Mid = both.
